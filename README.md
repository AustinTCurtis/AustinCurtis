# co2Package

`co2Package` is an R package designed to load, summarize, and visualize 
country-level annual CO2 emissions using data from the Our World in Data 
“Annual CO2 emissions per country” dataset.

This package was created as part of a project demonstrating R package 
development, defensive programming, debugging, GitHub version control, 
and R Markdown documentation.
